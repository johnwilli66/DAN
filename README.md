# DAN
**类**（class）
	类定义了
	属性（Attributes，存储数据的字段）和
	方法（Methods，执行操作的函数），规定了对象可执行的操作，以及不同对象之间的交互方式
 
**对象**object（数字，字符串，列表，字典）
	对象是类的实例，对象具有属性和方法
 
**变量**
	是对对象的引用
 
**类型**（Type）
	定义了对象的性质和行为
